Explorer-like plugin for walking through the pages

== Description ==

MS Windows Explorer-like plugin for administrating pages in WordPress. Useful for navigating through sites that have a complex page hierarchy.

== Installation ==

1. Upload `explore-pages` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
